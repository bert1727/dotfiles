# List of my dotfiles

# Requirements

```bash
sudo pacman -S stow
```

# Installation

```bash
git clone https://github.com/bert1727/dotfiles.git ~/dotfiles
```

## To stow all files

```bash
./stowing.sh
```

## To unstow all files

```bash
./unstowing.sh
```

## To restow all files

```bash
./restowing.sh
```
